# The Website That Slowly Loses Its Mind

## Idea
This is a deliberately useless and chaotic website that slowly becomes suspicious of the user.

At first it behaves like a normal webpage, but as the user interacts with it, the site begins questioning the user’s behaviour and existence on the page.

The goal of this project is not functionality, but humor and absurd interaction.

## Features

- Click-based story progression
- Fake loading progress
- Suspicious system analysis
- Mouse movement commentary
- A button that runs away from the user
- A fake "secret feature" reveal that roasts the user

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone the repository
2. Open `index.html` in a browser
3. Click **Start browsing**

## Why This Exists

Because sometimes the internet deserves something completely unnecessary.
